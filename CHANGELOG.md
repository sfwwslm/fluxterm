# 更新日志

本文档记录 FluxTerm 各版本已交付的用户可见功能与体验更新。

## 0.1.0-beta.8

本次更新聚焦于关于页能力增强、终端输入体验修复与终端配置细化，并同步统一了浮窗、子应用和配置界面的标题栏与控件表现。

### 新增

- **应用更新检查与安装**: 关于页新增更新检查与安装入口，可直接在应用内查看新版本并触发安装流程。
- **终端光标样式配置**: 新增全局终端光标样式设置，默认采用竖线光标，可按偏好切换显示效果。
- **终端分词器配置**: 新增终端分词器配置，支持在会话窗口设置中自定义单词边界，优化双击选词与按词跳转体验。

### 改进

- **关于页信息展示**: 重构关于页信息布局，补充系统诊断信息并支持一键复制，问题排查更高效。
- **终端快捷键策略**: 调整终端聚焦状态下的快捷键拦截规则，除少数高风险快捷键外，其余按键优先透传给 PTY，终端内交互更接近原生行为。
- **窗口标题栏统一**: 统一浮动窗口与子应用标题栏样式、右侧控件间距和头部结构，整体视觉与交互更一致。
- **配置界面收敛**: 统一 ConfigModal 下拉组件并清理遗留头部实现，设置界面结构更简洁一致。
- **按钮视觉收敛**: 收敛按钮样式分支，统一使用 ghost 风格，整体界面风格更一致。

### 修复

- **vim Esc 键行为**: 修复终端中 `Esc` 被宿主吞掉，导致 `vim` 无法退出插入模式的问题。
- **终端快捷键误拦截**: 修复终端聚焦时部分按键被错误拦截的问题，改善 `vim`、`less`、`htop` 等 TUI 场景可用性。
- **关于页控制台作用域**: 修复从关于页打开控制台时对子应用窗口不生效的问题。
- **会话窗口设置细节**: 修复分词器预设中空格分隔符被错误裁剪，以及编辑配置时窗口设置页签回跳的问题。

### 工程与流程

- **统一检查脚本**: 新增 `pnpm check` 前端统一检查命令，并升级 ESLint 工具链，提升日常开发校验一致性。

## 0.1.0-beta.7

本次更新聚焦于 SSH/SFTP 连接链路稳定性、日志可观测性与本地 shell 终端参数一致性，重点修复了无 SFTP 服务场景下的输入卡顿问题，并进一步收敛日志噪音。

### 改进

- **SFTP 检测去阻塞**: 调整会话命令分发策略，SFTP 能力检测不再阻塞终端输入处理，连接过程中的交互连续性显著提升。
- **SFTP 初始化超时保护**: 为 SFTP 初始化关键阶段增加超时控制，针对 OpenWrt 等未开启 SFTP 服务的场景可更快识别并回退到“不支持”状态。
- **SSH 认证埋点拆分**: 将 SSH 认证埋点按链路拆分为会话连接与资源监控两类事件，便于区分来源并提升排障效率。
- **日志降噪**: 下调 SFTP 性能与能力探测等高频非异常日志级别，减少常规使用中的信息干扰。
- **本地 Shell 参数打通**: 本地会话支持将终端类型与字符集配置按 `shellId` 持久化并生效，配置行为更可预期。

### 修复

- **无 SFTP 服务输入卡顿**: 修复连接无 SFTP 服务主机（如部分 OpenWrt）时，检测阶段终端短暂无法输入的问题。
- **macOS 本地 Shell TERM**: 修复 macOS 打包版本中本地 shell 的 `TERM` 缺失问题，恢复 Delete/Fn+Delete 等按键行为一致性。

### 工程与流程

- **Release 防重**: 工作流补充发布幂等保护，避免重复创建 GitHub Release。

## 0.1.0-alpha.5

本次更新重点补齐了 SSH 主机身份校验、OpenSSH 配置导入与凭据存储收敛，并继续提升终端信息展示与连接稳定性。

### 新增

- 新增 SSH 主机身份校验能力。
- 支持首次连接主机时确认主机身份，并在主机指纹变化时提示风险。
- 新增 SSH Host Key 校验策略，可在询问确认、严格阻断和关闭校验之间切换。
- 应用开始维护独立的 Host Key 信任记录，不与系统 `ssh` 命令的 `known_hosts` 混用。
- 新增从 OpenSSH `~/.ssh/config` 一键导入会话的能力，导入后的主机会统一归入 `OpenSSH` 分组。
- 导入支持读取常见的 `Host`、`HostName`、`Port`、`User`、`IdentityFile` 配置。
- 导入支持 `Host *` 默认值继承到后续具体主机块。

### 改进

- SSH 连接和资源监控链路都统一纳入主机身份校验流程，连接安全边界更清晰。
- 资源监控在主机未信任、探测失败或连接失败时，会返回明确状态，不再长时间停留在“资源检测中”。
- 资源监控会区分显示更具体的不可用原因，例如主机指纹未信任、远端系统不支持或资源采样失败。
- 资源监控的 SSH 采样执行方式调整为非 PTY，更适合机器采样场景，减少欢迎信息和交互式输出的干扰。
- 终端路径联动对提示符的识别规则进一步收紧，普通欢迎信息和状态行不再被误判为不支持的提示符。
- 会话列表中的 OpenSSH 导入结果会以更紧凑的摘要提示展示，并忽略值为 0 的结果项。
- OpenSSH 导入时，过长的会话名称会自动裁剪，并保留统一的长度约束。
- 状态栏新增代码行统计百分比显示，便于更直观地了解当前终端输出规模。

### 修复

- 修复主机已信任后资源监控仍可能卡在“资源检测中”的问题。
- 修复退出 SSH 会话后重新连接时，主机身份确认流程可能重复创建会话的问题。
- 修复取消主机身份确认后，确认弹窗仍会被重复触发的问题。
- 修复资源监控连接失败或被阻断时，前端状态无法正确结束的问题。
- 修复终端路径联动将欢迎信息误判为不支持提示符并输出无意义警告的问题。

### 安全性与存储

- 凭据存储格式收敛为统一的加密 token 形式，配置文件中的密码字段不再以展开 JSON 形式保存。
- 主机身份信任记录与应用配置目录统一收口，便于后续继续完善安全相关能力。

### 稳定性

- 升级底层 SSH 依赖，改善连接、保活与传输稳定性。
- 统一项目技术标识为 `fluxterm`，应用标识、配置目录和包名保持一致。

本次更新重点交付了终端工作区拆分、SFTP 目录级传输、历史命令与命令联想能力，并系统优化了会话标签与文件拖拽上传体验。

### 新增

- 终端支持在同一窗口内拆分多个工作区，可进行水平拆分、垂直拆分，并独立管理各区域中的会话。
- SFTP 文件管理器支持目录下载，传输面板会按项目数量展示批量任务进度，并正确统计批量目录任务的项目数量。
- SFTP 文件管理器支持将本地文件或目录直接拖拽上传到当前远端目录，并支持目录递归上传、目录删除等目录级操作。
- 新增“历史命令”组件，可实时显示当前输入行、浏览当前会话历史、搜索过滤并双击再次执行命令。
- 新增终端命令联想能力，可基于全局历史命令弹出候选列表，并支持跟随提示符定位、键盘选择填入与完整关闭交互。
- 会话设置新增命令联想开关，可按需关闭终端联想能力。

### 改进

- 会话标签支持右键菜单中的拆分、关闭、关闭右侧、关闭其他会话等工作区相关操作，终端编排更接近桌面终端使用习惯。
- 会话标签改为在各工作区内独立编号显示，分屏后每个区域都会从 1 开始编号。
- 会话标签仅在各可见工作区的当前活动会话上显示关闭按钮，长标题的截断、宽度和圆角样式也一并优化。
- 终端工作区在单 pane 场景下不再重复显示内部激活边框，避免与外层终端面板边框冲突。
- 本地 shell 列表中的 “Windows PowerShell” 调整为更简洁的 “PowerShell” 显示名称。
- 会话与分组名称增加长度与非空校验，减少列表换行和异常命名对界面的干扰。

### 修复

- 修复关闭当前 SSH 标签时，工作区仍有其他会话却短暂闪出“尚未连接主机”的问题。

## 0.1.0-alpha.3

本次更新重点完善了文件管理器的打开能力、会话管理的组织方式与编辑体验，并进一步稳定了浮动窗口和路径联动等桌面交互。

### 新增

- 文件管理器支持双击普通文件直接打开，本地文件可直接用系统默认程序或自定义编辑器打开，远端文件会先下载到本地缓存后再打开。
- 设置中新增默认编辑器配置，可通过选择程序的方式指定本机打开文件时优先使用的编辑器。
- 文件管理器“更多”菜单新增隐藏文件显示切换。
- 本地文件管理器“更多”菜单新增系统文件管理器入口，可直接打开当前本地目录。

### 改进

- 文件管理器改为更符合桌面习惯的交互：单击选中，双击打开。
- 文件管理器支持按系统语义识别本地隐藏文件，并对远端隐藏文件提供基础显示控制。
- 远端文件打开流程改为按实例目录缓存，每次打开都会生成独立本地副本，文件名保持原始名称。
- 会话列表调整为“分组在上、根级会话在下”的展示方式，未分组会话不再自动归入默认 SSH 分组。
- 会话列表为分组、本地 shell 和普通会话增加图标区分，信息层次更清晰。
- 会话编辑模态框重构为顶部类型切换、左侧菜单、右侧配置区与底部操作栏结构，为后续扩展更多 SSH 与会话选项预留空间。
- 通知横幅调整为顶部居中显示，视觉上更聚焦。

### 修复

- 修复文件管理器手动浏览目录后被旧终端路径强制跳回的问题，并消除联动图标在正常目录切换时的抖动。
- 修复重复下载同一路径文件时传输面板信息不刷新的问题。
- 修复浮动文件面板双击打开远端文件时失败的问题，统一由主窗口处理文件打开流程。
- 修复浮动窗口布局持久化异常，支持多个浮动窗口独立保存与恢复。
- 修复主窗口关闭时多个浮动窗口偶发无法一起关闭的问题。
- 修复组件区收起后仍占用实例，导致其他区域无法再次添加同一组件的问题。

## 0.1.0-alpha.2

本次更新重点补齐了终端交互细节、文件管理联动、资源监控与配置入口，并修复了一批会影响日常使用的稳定性问题。

### 新增

- 终端支持鼠标聚焦某一行，并可通过右键直接复制当前聚焦行或选中内容。
- 新增终端选中文本自动复制开关，可在会话设置中控制是否启用。
- 文件管理列表新增文件类型图标，提升目录与文件识别效率。
- 新增终端路径与文件管理器路径联动，支持在受支持的终端提示符下自动同步目录。
- 状态栏新增当前活动会话的资源监控信息，支持显示 CPU 与内存摘要，并可悬浮查看详情。
- 会话设置新增资源监控开关与采样间隔配置。

### 改进

- 新增终端滚动回溯配置，并支持对非法配置自动纠正。
- 状态栏新增当前终端缓冲区历史行数显示。
- 配置模态框改为统一的固定高度与内部滚动布局，配置项较多时不再撑高窗口。
- 会话标签补充最小点击宽度，短名称标签切换时不容易误点关闭按钮。
- 文件管理器与浮动窗口的关闭语义进一步统一，关闭浮动窗口后不再自动回收到主窗口。
- SFTP 初始化条件改为按需启用，并补充“检测中 / 不支持 / 已关闭”等更明确的状态提示。

### 修复

- 修复首个 SSH 会话在特定场景下横幅缺失、文件管理器长时间停留在检测状态的问题。
- 修复终端聚焦行、复制、空白区域命中与提示符闪烁等一系列交互问题。
- 修复终端与文件管理器路径联动在用户切换身份、浮动窗口和不支持终端场景下的状态异常。
- 修复 SFTP 在文件管理器不可见或浮动窗口关闭后仍被错误初始化的问题。
- 修复会话事件监听重复注册导致的 Tauri callback warning。
- 修复 macOS / Windows 菜单入口不一致导致的配置入口缺失问题。

## 0.1.0-alpha.1

首个 Alpha 版本，提供了可用的桌面终端、SSH 会话与 SFTP 文件管理基础能力。

### 已实现

- 支持通过 SSH 主机配置发起远程连接，并在桌面端进行终端交互。
- 支持本地 Shell 会话，便于在同一界面中使用本机终端。
- 支持多会话标签切换、断开连接与基础重连流程。
- 支持远端 SFTP 文件浏览，包括目录进入、返回 home 目录与列表刷新。
- 支持 SFTP 文件上传、下载，并显示传输进度与最近记录。
- 支持 SFTP 基础文件操作，包括新建目录、重命名与删除文件。
- 支持主机配置的新增、编辑、删除与分组管理。
- 支持工作区面板布局切换、拖拽重排、悬浮显示与底部传输面板联动。
- 支持快捷栏、状态区、事件/传输面板、关于窗口等基础桌面界面能力。

### 当前定位

- 该版本为 Alpha 阶段版本，重点在于完成 SSH、终端、SFTP 与桌面交互的基础闭环。
