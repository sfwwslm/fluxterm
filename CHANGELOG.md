# 更新日志

本文档记录 FluxTerm 各版本已交付的用户可见功能与体验更新。

## 0.1.0-alpha.2

本次更新重点补齐了终端交互细节、文件管理联动、资源监控与配置入口，并修复了一批会影响日常使用的稳定性问题。

### 新增

- 终端支持鼠标聚焦某一行，并可通过右键直接复制当前聚焦行或选中内容。
- 新增终端选中文本自动复制开关，可在会话设置中控制是否启用。
- 文件管理列表新增文件类型图标，提升目录与文件识别效率。
- 新增终端路径与文件管理器路径联动，支持在受支持的终端提示符下自动同步目录。
- 状态栏新增当前活动会话的资源监控信息，支持显示 CPU 与内存摘要，并可悬浮查看详情。
- 会话设置新增资源监控开关与采样间隔配置。

### 改进

- 新增终端滚动回溯配置，并支持对非法配置自动纠正。
- 状态栏新增当前终端缓冲区历史行数显示。
- 配置模态框改为统一的固定高度与内部滚动布局，配置项较多时不再撑高窗口。
- 会话标签补充最小点击宽度，短名称标签切换时不容易误点关闭按钮。
- 文件管理器与浮动窗口的关闭语义进一步统一，关闭浮动窗口后不再自动回收到主窗口。
- SFTP 初始化条件改为按需启用，并补充“检测中 / 不支持 / 已关闭”等更明确的状态提示。

### 修复

- 修复首个 SSH 会话在特定场景下横幅缺失、文件管理器长时间停留在检测状态的问题。
- 修复终端聚焦行、复制、空白区域命中与提示符闪烁等一系列交互问题。
- 修复终端与文件管理器路径联动在用户切换身份、浮动窗口和不支持终端场景下的状态异常。
- 修复 SFTP 在文件管理器不可见或浮动窗口关闭后仍被错误初始化的问题。
- 修复会话事件监听重复注册导致的 Tauri callback warning。
- 修复 macOS / Windows 菜单入口不一致导致的配置入口缺失问题。

## 0.1.0-alpha.1

首个 Alpha 版本，提供了可用的桌面终端、SSH 会话与 SFTP 文件管理基础能力。

### 已实现

- 支持通过 SSH 主机配置发起远程连接，并在桌面端进行终端交互。
- 支持本地 Shell 会话，便于在同一界面中使用本机终端。
- 支持多会话标签切换、断开连接与基础重连流程。
- 支持远端 SFTP 文件浏览，包括目录进入、返回 home 目录与列表刷新。
- 支持 SFTP 文件上传、下载，并显示传输进度与最近记录。
- 支持 SFTP 基础文件操作，包括新建目录、重命名与删除文件。
- 支持主机配置的新增、编辑、删除与分组管理。
- 支持工作区面板布局切换、拖拽重排、悬浮显示与底部传输面板联动。
- 支持快捷栏、状态区、事件/传输面板、关于窗口等基础桌面界面能力。

### 当前定位

- 该版本为 Alpha 阶段版本，重点在于完成 SSH、终端、SFTP 与桌面交互的基础闭环。
