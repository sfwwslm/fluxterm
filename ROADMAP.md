# FluxTerm 路线图

## 阶段 0：基础准备

- 定义 SSH/SFTP 引擎 API 边界。
- 完成 SSH 库选型并验证 Windows 支持。
- 明确配置与主机配置文件格式。

## 阶段 1：SSH 核心能力

- 连接生命周期（连接/重连/关闭）。
- 终端 I/O 流与窗口尺寸变更。
- 基础主机配置管理（新增/编辑/删除）。
- 主机指纹校验与持久化。

## 阶段 2：SFTP 能力

- 目录列表与基础元数据。
- 上传/下载与进度显示。
- 重命名/删除/创建目录。

## 阶段 3：GUI 与体验（优先）

- 会话标签页与快速连接。
- GUI 内的 SFTP 文件浏览器。
- 统一的错误处理与用户提示。

## 阶段 4：跨平台完善

- macOS/Linux 打包与冒烟测试。
- 平台钥匙串集成。
- 性能调优与内存分析。

## 待办与可选项

- SSH 密钥管理界面。
- 端口转发（本地/远端）。
- 会话历史与日志。
- CLI/TUI 客户端（GUI 完成后再评估）。
